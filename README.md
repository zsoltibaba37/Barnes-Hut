# Barnes-Hut

A C++ and SFML implementation of the Barnes-Hut algorithm for large gravity simulations.

---

## Installation

### 1. Prerequisites:

This project depends on the `SFML` library.

- **Debian/Ubuntu:**

  ```bash
  sudo apt update
  sudo apt install libsfml-dev
  ```

### 2. Clone the repo:
  ```bash
  git clone https://github.com/milcsu09/Barnes-Hut.git
  ```

### 3. Build:
  ```bash
  cd Barnes-Hut
  make
  ```

---

## Controls

- `W` `A` `S` `D`: Move camera
- `Mouse Scroll`: Zoom in/out
- `Tab`: Toggle position interpolation
